EPA.gov redesign of header navigation and overall layout refinement.

THE PROBLEM:

Users are having difficulty finding information via the current menu navigation structure.

THE SOLUTION:

Improve the dropdown menu selection process to add more tiers, so that a user can find an article or page without navigating multiple pages.

MY ROLE:

UI/UX Researcher/Designer

TOOLS:

Figma, Adobe Xd, Miro, Zoom, optimalworkshop.com, index cards

User Interface Analysis

Users were tasked with finding the page about mold cleanup for their home.

Research found that users were not clear on where to click to find the information that they were looking for. The navigation category “Environmental Topics” didn’t suggest a clear path.

All users considered to the search box. The path tended to be a little convoluted when they did click through from the main category dropdown.

A cart sort was conducted on optimalworkshop.com to determine what the best category title should be. A total of 10 users participated.

Findings:

“Home Health” edged out Healthy Home as the top choice for the new category in the main navigation bar
